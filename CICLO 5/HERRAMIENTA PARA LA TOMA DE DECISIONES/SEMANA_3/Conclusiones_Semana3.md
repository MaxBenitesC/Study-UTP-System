# Conclusiones - Semana 3: Introducción a Excel

1. Microsoft Excel es un programa informático que permite procesar información para generar reportes visuales útiles para la toma de decisiones.

2. Las celdas relativas y absolutas permiten mantener valores cuando se autocompletar filas y/o columnas.

3. Microsoft Excel se ha convertido en una de las principales herramientas informáticas en el mundo empresarial.
