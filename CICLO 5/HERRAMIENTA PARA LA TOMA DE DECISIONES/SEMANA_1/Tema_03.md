# Tema 03: Enterprise Resource Planning (ERP)

## Definición

Un **ERP** (Sistema de Planificación de Recursos Empresariales) es una herramienta informática de gestión de datos en la que se integra y automatiza la información de todas las áreas de una empresa. Sirve para tener mayor control operativo y ahorrar tiempo y costos.

---

## Características

- Base de datos común que soporta todas las aplicaciones del sistema
- Opera prácticamente en tiempo real
- Sistema completamente integrado
- Apariencia consistente en todos los módulos
- Integración elaborada de aplicaciones
- Implementación flexible: local, en la nube o como SaaS (Software como Servicio)

---

## Usos principales

Integra las áreas clave de la empresa, destacando:

- **Producción**
- **Logística**
- **Contabilidad**

---

---

## Resumen

El ERP es un único programa con acceso a una base de datos centralizada que funciona como un sistema integrado, aunque puede contener varios módulos especializados. Entre los principales proveedores se encuentran: **SAP**, **FIS Global**, **Oracle** y **Fiserv**.
