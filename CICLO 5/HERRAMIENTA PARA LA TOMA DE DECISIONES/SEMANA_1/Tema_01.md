# Tema 01: Sistemas de Información

## Definición

Un **sistema de información** es una herramienta informática que se implementa dentro de una organización o empresa para gestionar la información y dar soporte a la toma de decisiones.

---

## Tipos Principales

| Sistema | Descripción breve |
|---|---|
| **Sistemas de Información de Gestión (SIG)** | Proveen informes y datos consolidados para apoyar la gestión operativa y táctica. |
| **Sistemas de Gestión Documental** | Administran y organizan documentos digitales dentro de la organización. |
| **Sistemas de control de procesos de negocio** | Monitorean y controlan los procesos internos de la empresa. |
| **Sistemas de apoyo a la toma de decisiones (SAD)** | Facilitan el análisis de datos para la toma de decisiones complejas. |
| **Sistemas de Información Ejecutiva (SIE)** | Brindan información estratégica resumida para la alta dirección. |
| **Sistemas de Gestión de Almacenes** | Controlan y optimizan las operaciones de almacenamiento e inventario. |
| **Sistemas de procesamiento de transacciones (SPT)** | Registran y procesan las transacciones diarias de la organización. |
| **Sistemas de colaboración empresarial** | Facilitan la comunicación y trabajo conjunto entre equipos. |

---

## Resumen

Existen múltiples tipos de sistemas de información, cada uno con diferentes características y propósitos. Las empresas deben evaluar sus necesidades y elegir los sistemas más adecuados según sus características organizacionales, con el objetivo de proveer la información necesaria para respaldar eficazmente el proceso de toma de decisiones.
