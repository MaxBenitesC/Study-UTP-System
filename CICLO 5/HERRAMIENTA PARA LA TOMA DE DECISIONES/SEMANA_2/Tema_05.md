# Tema 05: Microsoft Excel

## Definición

**Microsoft Excel** es una herramienta avanzada de análisis y visualización de datos. Es un procesador de hojas de cálculo desarrollado por Microsoft, disponible para múltiples sistemas operativos.

---

## Sistemas operativos compatibles

- Windows
- macOS
- Android
- iOS

---

## Características principales

| Característica | Descripción |
|---|---|
| **Cálculo** | Herramientas matemáticas y financieras para procesar datos numéricos. |
| **Gráficas** | Efectos visuales y gráficos atractivos y fáciles de analizar. |
| **Tablas dinámicas** | Resumen y análisis interactivo de grandes volúmenes de datos. |
| **Macros (VBA)** | Lenguaje Visual Basic para Aplicaciones que permite automatizar tareas. |
| **Compatibilidad externa** | Conexión con bases de datos, servicios web y otras fuentes para recopilar información. |
| **Power Pivot** | Complemento para procesamiento y modelado avanzado de datos. |

---

## Usos en modelamiento empresarial

- Modelos de tendencias lineales y no lineales
- Modelos de evaluación de proyectos de inversión
- Modelos de segmentación de mercados
- Modelos de gestión integral de negocios

---

## Resumen

Excel dispone de herramientas de búsqueda y optimización que permiten a las empresas tomar decisiones acertadas orientadas a **reducir costos o maximizar ganancias**. Sus hojas de cálculo permiten cuantificar e integrar información de diversas fuentes, presentándola de forma visual e intuitiva para cualquier usuario, sin necesidad de ser experto en informática.
