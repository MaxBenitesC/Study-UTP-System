# Avance de Proyecto Final 1 — Programación Web 2026

## Descripción General

Construir un sitio web multi-página orientado a la **reducción del desperdicio de alimentos**, permitiendo que restaurantes o negocios oferten comidas próximas a vencer a precios accesibles.

> Tecnología permitida: **solo HTML puro**. No se permite `<style>`, atributo `style` ni archivos CSS externos.

---

## Páginas requeridas

| Archivo | Descripción |
|---|---|
| `login.html` | Iniciar sesión |
| `registrar.html` | Registrar usuario |
| `index.html` | Inicio / Home |
| `menu.html` | Menú de comidas |
| `form_pedido.html` | Formulario de pedido |
| `pedidos.html` | Tabla de pedidos realizados |
| `ofertas.html` | Ofertas especiales |

---

## Requisitos por página

### 1. `login.html` — Iniciar Sesión
- [ ] Formulario con campos: **Usuario** y **Contraseña**
- [ ] Botón **"Ingresar"**

---

### 2. `registrar.html` — Registrar Usuario
- [ ] Formulario con campos:
  - Nombre completo
  - Correo electrónico
  - Contraseña
  - Confirmar contraseña
- [ ] Botón **"Registrar"**
- [ ] Enlace hacia `login.html` *(opcional, depende del alumno)*

---

### 3. `index.html` — Inicio
- [ ] Nombre de la página web / logo
- [ ] Mensaje de bienvenida al usuario
- [ ] Descripción del propósito (rescate de alimentos)
- [ ] Navegación hacia:
  - Menú (`menu.html`)
  - Pedido (`form_pedido.html`)
  - Ofertas (`ofertas.html`)
- [ ] Footer con contacto o redes sociales
- [ ] Enlaces de navegación hacia Menú y Pedido

---

### 4. `menu.html` — Menú de Comidas
- [ ] Lista de productos disponibles
- [ ] Tabla con columnas:
  - Nombre del producto
  - Descripción
  - Precio original
  - Precio con descuento
- [ ] Botón **"Añadir al Pedido"** o **"Añadir a carrito de compra"**

---

### 5. `form_pedido.html` — Formulario de Pedido
- [ ] Formulario con campos:
  - Nombre del cliente
  - Dirección
  - Teléfono
  - Selección de platos (checkbox o select)
  - Cantidad
- [ ] Botón **"Enviar Pedido"**
- [ ] Enlace para regresar al Menú o Inicio

---

### 6. `pedidos.html` — Tabla de Pedidos Realizados
- [ ] Tabla con columnas:
  - Número de pedido
  - Cliente
  - Productos seleccionados
  - Cantidad
  - Total
  - Estado: `Enviado` / `En Proceso` / `Preparando`
- [ ] Enlace para regresar al Menú o Inicio

---

### 7. `ofertas.html` — Ofertas Especiales
- [ ] Lista o tabla de productos con mayor descuento
- [ ] Indicador visual de urgencia (ej: *"Últimas unidades"*)

---

## Restricciones técnicas

- Solo se usa **HTML puro**
- **Prohibido** usar `<style>`, atributo `style` o archivos `.css`
- Si se detectan estilos CSS → **descuento en la calificación**
