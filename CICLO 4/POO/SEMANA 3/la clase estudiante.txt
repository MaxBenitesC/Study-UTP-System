conceptos previos:
Programación Orientada a Objetos: parecida a la forma de percibir el mundo real por parte de los seres humanos: objetos que se relacionan (se comunican mediante mensajes=invocación a métodos) 
Clase: plantilla para la creación de objetos.
Instancia: objeto creado
MIEMBRO Atributo / campo: variables de instancia (forman parte de los datos de un objeto)
MIEMBRO Operación / método: actividades que realiza un objeto
MIEMBROS del objeto (no static) y MIEMBROS de la clase(static)
CONSTRUCTOR: método que se invoca (con new) para instanciar un objeto. Tiene el mismo nombre que la clase a la que pertenece y no tiene tipo de retorno. 
SOBRECARGA de métodos: tipo de polimorfismo DENTRO de la clase, donde se diferencian por el número o tipo de parámetros. 
SOBRESCRITURA con toString(): tipo de polimorfismo ENTRE clases (super clase=base y sub clase=derivada). Sirve para desplegar (imprimir los valores los campos) un objeto.
VISIBILIDAD private(accesible solo DENTRO de SU clase) y public(accesible en TODA la aplicación)
THIS: referencia al OBJETO actual
RELACIÓN (1 de 4) DE HERENCIA: "jerarquía" donde la instancia de la subclase incialmente es un gemelo de su superclase. Luego se "extiende" (añaden ESTADO y COMPORTAMIENTO propios del la instancia hija.
MODELO: descripción, resumen, diagrama, bosquejo, construcción que es una ABSTRACCIÓN que facilita la solución de un problema COMPLEJO. Ejm: maqueta de los arquitectos.
UML: estándar de modelamiento para la orientación a objetos

package demomain;
public class Estudiante {
   // Atributos privados
   private String nombre;
   private int codigo;
   private boolean sexo; // true = masculino, false = femenino
   private static int cantidadEstudiantes = 0;
   // Constructores sobrecargados
   public Estudiante() {
       this.nombre = "Sin nombre";
       this.codigo = 0;
       this.sexo = true; // por defecto masculino
       cantidadEstudiantes++;
   }
   public Estudiante(String nombre) {
       this.nombre = nombre;
       this.codigo = 0;
       this.sexo = true;
       cantidadEstudiantes++;
   }
   public Estudiante(String nombre, int codigo) {
       this.nombre = nombre;
       this.codigo = codigo;
       this.sexo = true;
       cantidadEstudiantes++;
   }

   public Estudiante(String nombre, int codigo, boolean sexo) {
       this.nombre = nombre;
       this.codigo = codigo;
       this.sexo = sexo;
       cantidadEstudiantes++;
   }
   // Getters y Setters
   public String getNombre() {  return nombre;  }
   public void setNombre(String nombre) {   this.nombre = nombre;   }
   public int getCodigo() {  return codigo;   }
   public void setCodigo(int codigo) {   this.codigo = codigo;  }
   public boolean isSexo() {  return sexo; }
   public void setSexo(boolean sexo) {  this.sexo = sexo;    }
   public static int getCantidadEstudiantes() {
       return cantidadEstudiantes;
   }

   // Método toString sobrescrito
   @Override
   public String toString() {
       return "Estudiante{" +
               "nombre='" + nombre + '\'' +
               ", codigo=" + codigo +
               ", sexo=" + (sexo ? "Masculino" : "Femenino") +
               '}';
   }
}
